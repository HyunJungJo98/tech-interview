# React

- Redux, Context API 등 상태 관리 라이브러리가 다양하며, 상황에 맞게 선택할 수 있는 유연성을 제공
- 자바스크립트 문법을 쓰고 싶은 대로 사용이 가능하여 자유도가 높음

# Vue

- Vuex를 통해 일관된 상태 관리 패턴을 제공
- 제공해주는 문법에 한해서만 구현이 가능
